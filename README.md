# TaskGroupBot

Telegram-бот для командного управления задачами в групповом чате.
Бот позволяет добавлять задачи в общий список, просматривать их, редактировать
(назначать ответственного, дедлайн, статус) и экспортировать в CSV.

## Возможности

- Добавление задач в общий список (команда /add)
- Просмотр всех задач в чате (команда /list)
- Экспорт задач в CSV-файл (команда /list_csv)
- Редактирование задач: назначение ответственного, дедлайн, статус (команда /edit)
- Автоматическая простановка статуса «просрочено» при наступлении дедлайна
  (проверка раз в 1 час, уведомление в чат с тегом ответственного)
- Работает только в групповых чатах (личные сообщения игнорируются)
- Все задачи сохраняются с меткой времени (GMT+3, Москва)

## Стек технологий

- **Python 3.11+**
- **aiogram 3.x** — асинхронный фреймворк для Telegram-ботов
- **aiosqlite** — асинхронная обёртка над SQLite
- **python-dotenv** — загрузка переменных окружения из .env
- **zoneinfo** — работа с часовыми поясами (стандартная библиотека)

## Команды бота

| Команда | Описание | Пример |
|---------|----------|--------|
| /start | Приветствие со списком команд | /start |
| /add | Добавить задачу (бот запросит текст) | /add -> текст задачи |
| /list | Показать все задачи | /list |
| /list_csv | Экспорт задач в CSV-файл | /list_csv |
| /edit | Редактировать задачу (inline-кнопки) | /edit -> ID -> выбор поля |

### Команда /edit — подробности

Поток через inline-кнопки:

1. `/edit` — бот просит ввести ID задачи
2. Ввод ID — бот показывает кнопки: «Назначить ответственного», «Установить дедлайн», «Сменить статус»
3. **Ответственный** — кнопки со списком участников группы (которые ранее взаимодействовали с ботом) + «Снять»
4. **Дедлайн** — кнопки: «Сегодня», «Завтра», «+3 дня», «+7 дней» (с реальной датой) + «Снять»
5. **Статус** — кнопки: «В работе», «Выполнено» (статус «Просрочено» устанавливается автоматически)

### Автопросрочка

- Фоновая проверка раз в 1 час
- Задачи со статусом «в работе» и истёкшим дедлайном автоматически получают статус «просрочено»
- В чат отправляется уведомление: `Задача #5 просрочена! @ivan — дедлайн был 2024-01-20`

## Пример работы

```
Вы:  /add
Бот: Введите текст задачи:
Вы:  Подготовить презентацию к пятнице
Бот: Задача #1 добавлена: Подготовить презентацию к пятнице

Вы:  /edit
Бот: Введите ID задачи для редактирования:
Вы:  1
Бот: [Кнопки: Ответственный | Дедлайн | Статус]
     -> Нажимаем «Установить дедлайн»
Бот: [Кнопки: Сегодня (2024-01-15) | Завтра (2024-01-16) | +3 дня | +7 дней | Снять]
     -> Нажимаем «Завтра (2024-01-16)»
Бот: Задача #1: дедлайн установлен на 2024-01-16

Вы:  /list
Бот: Список задач:
      1. Подготовить презентацию к пятнице — ivan, 2024-01-15T14:30
         Дедлайн: 2024-01-16 | Статус: в работе

Вы:  /list_csv
Бот: [Отправляет файл tasks.csv]
```
