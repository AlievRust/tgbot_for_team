# Proposal: Базовый функционал Telegram-бота TaskGroupBot

## Описание изменения
Создание MVP Telegram-бота для командного управления задачами в групповом чате.
Реализация базовых команд: /start, /add, /list, /list_csv.

## Мотивация
Команде до 100 человек нужен простой инструмент для фиксирования задач
напрямую в групповом чате Telegram, без переключения в сторонние приложения.

## Область применения
- Только групповые чаты (личные сообщения игнорируются)
- Один общий список задач на группу
- Все участники могут добавлять и просматривать задачи

## Команды
| Команда | Описание |
|---------|----------|
| /start | Приветственное сообщение с описанием доступных команд |
| /add | Запрос текста задачи → сохранение в БД (FSM: двухшаговый процесс) |
| /list | Вывод всех задач в сообщении (формат: ID. текст — @user, дата) |
| /list_csv | Экспорт всех задач в CSV-файл → отправка документа в чат |

## БД-схема (таблица tasks)
| Поле | Тип | Описание |
|------|-----|----------|
| id | INTEGER PRIMARY KEY AUTOINCREMENT | Уникальный ID задачи |
| text | TEXT NOT NULL | Текст задачи |
| user | TEXT NOT NULL | Имя пользователя (telegram username или display name) |
| created_at | TEXT NOT NULL | Дата/время создания (GMT+3, формат ISO 8601) |

## Принятые решения
- Часовой пояс жёстко зафиксирован: Europe/Moscow (GMT+3)
- FSM для /add: бот запрашивает текст → пользователь отвечает текстом → сохранение
- При пустом списке задач /list и /list_csv сообщают об этом
- CSV-кодировка: UTF-8 с BOM (для корректного открытия в Excel)
