## Назначение
Telegram-бот для командного управления задачами в групповом чате.
Позволяет добавлять, просматривать, редактировать задачи и экспортировать их в CSV.
Поддерживает назначение ответственного, дедлайнов и автоматическую простановку статуса «просрочено».

## Цели
- Простой инструмент для фиксирования задач без переключения в сторонние приложения
- Автоматизация контроля сроков через дедлайны и автопросрочку
- Назначение ответственных прямо в чате

## Ограничения
- Только групповые чаты (личные сообщения игнорируются)
- Один общий список задач на группу
- Часовой пояс: Europe/Moscow (GMT+3)
- Дедлайн — только дата, без времени
- Статус «просрочено» устанавливается автоматически, вручную — нельзя
- Список участников группы формируется по факту взаимодействия с ботом (ограничение Telegram API)
