Идея: создать проект телеграм0бота на Python для командной работы: добавление задач в общий список
Бот должен принимать команды: /start, /add, /list, /list_csv
Бот получает сообщение и записывает задачи в БД (sqlite)
Пример работы:
/start - бот приветствует
/add - бот запрашивает текст задачи и записывает в таблицу id задачи, text, user, created_at (GMT+3)
/list - бот выводит в ответном сообщении список всех задач
/list_csv - бот экспортирует все задачи в csv файл

Структура проекта: точка входа main.py, для разных функций - отдельные пакеты
Обязательно подробные комментарии в коде
credencials в .env файле
создать .gitignore
